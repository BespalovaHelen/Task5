# Task 5: yacc

Построение синтаксических анализаторов с помощью **yacc** (bison).

## Содержание

| Файл   | Описание                                |
|--------|-----------------------------------------|
| `1.y`  | Вычисление **протяжения** скобочной системы |
| `2.y`  | Калькулятор арифметических выражений     |
| `Makefile` | Сборка и очистка проекта             |

### Примеры работы

1. type a string:

> ()()()
extent: 3

2.Калькулятор

enter expression:

> 2 + 3 * 4
result: 14

> (10 - 2) / 4
result: 2
